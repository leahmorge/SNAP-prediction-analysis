# SNAP Benefits Geospatial Analysis

## Overview
This project analyzes geographic patterns in SNAP (Supplemental Nutrition Assistance Program) participation and related socioeconomic indicators using QGIS and spatial data.

## Objectives
- Map and visualize SNAP participation patterns
- Analyze relationships between benefits, housing costs, and geographic variation
- Demonstrate geospatial analysis and data visualization skills

## Tools & Technologies
- QGIS
- Python
